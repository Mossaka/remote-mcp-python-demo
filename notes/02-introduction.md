# Section 1: Project Setup and Basic MCP Server

## Demo Commands

```bash
# 1. Initialize a new Python project with uv
uv init remote-mcp-demo
cd remote-mcp-demo
```

```bash
# 2. Run the default hello script to verify setup
uv run hello.py
```

```bash
# 3. Install Python MCP SDK
uv add "mcp[cli]"
```
